Help Information

Frequently Asked Questions
Why can't an employee log in?
To log into the Admin website, you should be a Manager.
How can I change my password?
You can change your password using the edit button next to your user information.
How can I find a user?
You can search a user by their name or username using the search button on top right
Why can't I find a user?
You can search a user using the search feature and also make sure you check all pages
Do I have to worry about assigning a unique username?
While creating a user, it will display a warning if the username is already taken.
How can I see my own information?
You can view your own user information by clicking the dropdown next to your username and clicking on user info
